# Tutor-Me

## Serverless Deployment

### dependencies:
- aws-sdk: "^2.790.0",
- body-parser: "^1.19.0",
- express: "^4.17.1",
- password-hash: "^1.2.2",
- serverless-http: "^2.6.0"
- cors: "^2.8.5",


`$ npm install -g serverless`

`$ npm i --save aws-sdk body-parser express password-hash serverless-http cors`

**Make sure to set up AWS configuration**

`$ serverless deploy`
